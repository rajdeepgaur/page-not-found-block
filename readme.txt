A WordPress block for the "Page Not Found" (404) template that retrieves and displays the last part of the permalink slug.

== Description ==

**Page not found block** is a simple yet powerful WordPress block designed for **custom 404 pages**.  
It dynamically extracts and displays the last segment of the requested URL when a visitor lands on a **404 error page**.  

✔ **Retrieves and displays the last part of the permalink (slug)**  
✔ **Should be used only on the 404 template** for better UX  
✔ **Can be styled using block settings or custom CSS**  
✔ **Compatible with Full Site Editing (FSE) and classic themes**  

This block is particularly useful for **custom 404 pages** to inform users what they were looking for and guide them back to relevant content.
